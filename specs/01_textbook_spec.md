# 第I部 一様等方宇宙と熱史

## 第1章 FLRW計量と膨張宇宙（14p）

**目標**: 一様等方性の仮定だけからFLRW計量を書き下し、Einstein方程式からFriedmann方程式を導き、ΛCDM背景宇宙の $H(a)$, $\eta(a)$, 各距離・地平線を計算可能にする。

**節構成**
1.1 宇宙原理と観測的根拠（CMB等方性 $10^{-5}$、銀河分布）— 2p
1.2 FLRW計量の導出（最大対称3次元空間の計量から。$k=0,\pm1$。本書は以後 $k=0$ を主筋とし、曲率は第13章で復活）— 3p
1.3 測地線と赤方偏移（$1+z = a_0/a$ の導出。光子4元運動量 $p^\mu$ の定義と $p \propto 1/a$）— 2p
1.4 Einstein方程式 → Friedmann方程式（導出契約S1.1）— 3p
1.5 構成要素とスケーリング（$\rho_r \propto a^{-4}$, $\rho_m \propto a^{-3}$, $\rho_\Lambda$ const。連続の式から導出。$\Omega_i$ 定義、fiducial値表）— 2p
1.6 共形時間・粒子的地平線・ハッブル地平線（$\eta(a)$ の積分表式、放射優勢/物質優勢での解析解 $a\propto\eta, a\propto\eta^2$、等密度時 $a_{\rm eq} = \Omega_{r0}/\Omega_{m0}$ の導出と数値 $z_{\rm eq}\approx3400$）— 2p
1.7 距離（共動距離、角径距離 $d_A$。**$C_\ell$ の射影で使う共動角径距離 $\chi(z_*)\approx 13.9$ Gpc をここで数値計算**）— 2p

**導出契約**
- S1.1: FLRW計量（入力）→ Christoffel記号全成分 → Ricciテンソル → $H^2 = \frac{8\pi G}{3}\rho - \frac{k c^2}{a^2}$ および $\frac{\ddot a}{a} = -\frac{4\pi G}{3}(\rho + 3p/c^2)$（出力）。Christoffelは付録Aに全成分表を置き、本文では非零成分の代表計算1つを示して残りは表参照とする。
- S1.2: $\nabla_\mu T^{\mu\nu}=0$（入力）→ $\dot\rho + 3H(\rho + p/c^2) = 0$（出力）→ 各成分スケーリング。
- S1.3: ヌル測地線（入力）→ $1+z = 1/a$（出力）。
- S1.4: $\mathcal{H}(x) = H_0\sqrt{\Omega_{m0}e^{-x} + \Omega_{r0}e^{-2x} + \Omega_{\Lambda0}e^{2x}}$ と $\eta(x) = \int_{-\infty}^{x} \frac{c\,dx'}{\mathcal{H}(x')}$（数値実装第14章への接続点として明示）。

**図版**（全てNB1で生成）

- Fig 1.2: $\mathcal{H}(x)/H_0$ と、比較線 $\propto e^{-x}, e^{-x/2}, e^{x}$
- Fig 1.3: $\eta(a)$ と地平線スケール $c/\mathcal{H}$、$k=10,100,1000\,H_0/c$ の地平線進入時刻を図示（スライドp23–25の解釈の伏線）

**演習例**: 物質優勢宇宙での $\eta \propto a^{1/2}$ の導出/ $z_{\rm eq}$ の $\Omega_m h^2$ 依存/ NB1で $\Omega_\Lambda$ を変えて $\eta_0$ がどう変わるか。

## 第2章 熱平衡宇宙の熱史（14p）

**目標**: 平衡統計力学を膨張宇宙に適用し、「温度 $T\propto 1/a$ の黒体放射」「ニュートリノ脱結合と $T_\nu/T_\gamma=(4/11)^{1/3}$」「相互作用率 vs 膨張率による脱結合の一般基準」を確立する。CMBが黒体である理由をここで完結させる。

**節構成**
2.1 分布関数と熱力学量（$f(\mathbf{p})$, $n,\rho,p$ の積分表式。Bose/Fermi。相対論極限・非相対論極限の評価、$g_*$）— 3p
2.2 膨張宇宙での平衡維持条件 $\Gamma \gtrsim H$（次元解析による電弱期〜再結合期のスケール感）— 2p
2.3 エントロピー保存と $T(a)$（$s \propto a^{-3}$ の導出、$g_{*s}$）— 2p
2.4 ニュートリノ脱結合と電子陽電子対消滅（$T_\nu/T_\gamma$ の導出契約S2.1。$N_{\rm eff}=3.046$ の由来は【発展】ボックスで1段落）— 3p
2.5 黒体性の維持: 自由膨張下で黒体スペクトルが黒体のまま $T\propto1/a$ で冷えることの証明（Liouville方程式で $f(p)$ 不変 → プランク分布の形状不変性）。COBE/FIRASへの言及 — 2p
2.6 トムソン散乱とコンプトン散乱（微分断面積 $\frac{d\sigma_T}{d\Omega} = \frac{3\sigma_T}{16\pi}(1+\cos^2\theta)$ を提示し、トムソン極限の妥当性 $h\nu \ll m_e c^2$ を数値で確認。エネルギー移行が $O(h\nu/m_ec^2)$ で抑制されること、したがって再結合期近傍では散乱は方向のみ変える、という第6章の衝突項の物理的基礎をここに置く）— 2p
2.7 光子・バリオン・電子系の熱平衡（クーロン散乱による $T_b \simeq T_\gamma$ の維持、スライドp8の相互作用図をここで提示し本書全体の登場人物を確定: 計量・光子・ニュートリノ・バリオン(=陽子+電子)・CDM・Λ）— 2p

**導出契約**
- S2.1: エントロピー保存（入力）→ $T_\nu/T_\gamma = (4/11)^{1/3}$、$\Omega_{\nu0} = N_{\rm eff}\frac{7}{8}(4/11)^{4/3}\Omega_{\gamma0}$（出力）。
- S2.2: $\Omega_{\gamma0}h^2 = 2.47\times10^{-5}$ を $T_{\rm CMB}=2.7255$ K から数値導出。
- S2.3: Liouville → 黒体形状不変性。

## 第3章 再結合とCMBの誕生（14p）

**目標**: 電離度 $X_e(a)$ をSaha→Peeblesで定量計算し、光学的厚み $\tau$ と可視度関数 $\tilde g$ を定義して「最終散乱面」を数式として確立する。本書の縦糸である $\tau(\eta), \tilde g(\eta)$ はここで完成する。

**節構成**
3.1 水素再結合の素過程（直接再結合光子の問題、2光子崩壊とLyα脱出 — 物理的説明）— 2p
3.2 Saha方程式（導出契約S3.1。平衡が破れる理由）— 3p
3.3 Peebles方程式（導出契約S3.2。3準位原子、各係数 $\alpha^{(2)}, \beta, \beta^{(2)}, \Lambda_{2s\to1s}=8.227\,{\rm s}^{-1}$, Lyα脱出率 $\Lambda_\alpha$ の物理的意味。完全導出は付録D、本文は方程式の構造と各項の意味＋律速の同定）— 4p
3.4 光学的厚みと可視度関数（$\tau(\eta) = \int_\eta^{\eta_0} n_e \sigma_T a\,d\eta'$、$\tilde g = -\tau' e^{-\tau}$、$\int \tilde g\,dx = 1$ の証明。「最後に散乱された時刻の確率密度」という解釈の導出契約S3.3）— 3p
3.5 数値結果（$X_e(x)$: Saha vs Peebles 比較、$z_* \approx 1090$（$\tau=1$ 定義）、可視度関数の幅 $\Delta z \approx 80$ → 最終散乱「面」に厚みがあること。この厚みが第9章の減衰と第10章の積分に効く伏線）— 2p
3.6 再電離の予告（$z_{\rm re}\sim8$ で $X_e\to1$ に戻ること、$\tau_{\rm re}\approx0.05$。詳細は第13章）— 1p
3.7 自由電子の減少と光子平均自由行程の発散（$\lambda_{\rm mfp} = 1/(n_e\sigma_T a)$ の爆発的増大の図示 → 「脱結合」の定義）— 1p

**導出契約**
- S3.1: 化学平衡 $\mu_p+\mu_e=\mu_H$（入力）→ $\frac{X_e^2}{1-X_e} = \frac{1}{n_b}\left(\frac{m_e k_B T}{2\pi\hbar^2}\right)^{3/2} e^{-\epsilon_0/k_BT}$（出力、$\epsilon_0=13.6$ eV）。「なぜ $T\sim\epsilon_0$ でなく $T\sim\epsilon_0/40$ で再結合するか」をバリオン光子比 $\eta_b\sim6\times10^{-10}$ から定量的に説明。
- S3.2: Peebles方程式 $\frac{dX_e}{dx} = \frac{C_r(T_b)}{H}\left[\beta(T_b)(1-X_e) - n_H\alpha^{(2)}(T_b)X_e^2\right]$ の各係数定義式一式（`03_numerics_spec.md` §2 と完全一致させること）。
- S3.3: 「最後の散乱が $(\eta,\eta+d\eta)$ で起こる確率 $= \tilde g\,d\eta$」の導出。

## 第4章 計量摂動とゲージ（12p）

**目標**: 摂動の SVT 分解の必要最小限を示し、ゲージ自由度を理解した上で conformal Newtonian ゲージを**選択する理由つきで**固定する。

**節構成**
4.1 なぜ摂動論か（$10^{-5}$ のゆらぎ → 線形化の正当性。どこまで線形が使えるかの定量的見積り）— 1p
4.2 計量摂動の一般形とSVT分解（スカラー4自由度のみ本文で展開。ベクトル・テンソルは【発展】で減衰/重力波に1p言及）— 3p
4.3 ゲージ変換（無限小座標変換 $x^\mu \to x^\mu + \xi^\mu$ による $\delta g_{\mu\nu}$, $\delta\rho$, $v$ の変換則。導出契約S4.1。「密度ゆらぎはゲージ依存」の具体例）— 4p
4.4 conformal Newtonianゲージの固定と計量 $ds^2 = a^2(\eta)[-(1+2\Psi)d\eta^2 + (1+2\Phi)\delta_{ij}dx^idx^j]$（残留ゲージ自由度がないことの確認）— 2p
4.5 ゲージ不変量 $\mathcal{R}$（共動曲率ゆらぎ）の定義（第7章の初期条件で使うため。Bardeen変数との関係は付録A）— 2p
4.6 流体変数の定義（$\delta_i, v_i$、速度の符号・成分規約をここで一意に固定: $T^0{}_i = (\rho+p)v_i$, スカラー速度 $v$ は $v_i = -\frac{ik_i}{k}v$ と定義 — `03_numerics_spec.md` と一致させる）— 2p

**導出契約**
- S4.1: $\xi^\mu$ 変換（入力）→ $\Psi, \Phi, \delta, v$ の変換則（出力）→ Newtonianゲージ固定。
- S4.2: $\mathcal{R} = \Phi - \frac{\mathcal{H}}{k}v$（本書規約での符号を確定し、超地平線で保存することの証明は第7章S7.2へ前方参照）。

## 第6章 ボルツマン方程式（20p）— 本書の心臓部

**目標**: 一般のボルツマン方程式 $\frac{df}{d\lambda} = C[f]$ から出発し、(a) 光子: トムソン衝突項を持つ温度ゆらぎの階層方程式、(b) CDM: 無衝突流体方程式、(c) バリオン: 運動量交換項付き流体方程式、(d) ニュートリノ: 無衝突階層、の全てを導出する。スライドp13–16の方程式群が全て「導出済みの式」になる。

**節構成**
6.1 位相空間とLiouville演算子（測地線方程式から $\frac{dp}{d\eta}$, $\frac{d\hat p^i}{d\eta}$ を摂動1次で計算。導出契約S6.1。計量摂動中の光子のエネルギー変化＝重力赤方偏移とポテンシャル時間変化、の物理的注釈）— 4p
6.2 光子分布関数の摂動と輝度関数（$f = \bar f(p/(1+\Theta))$ 型のパラメータ化、$\Theta(\eta,\mathbf{x},\hat p)$ の定義、エネルギー積分で $p$ 依存性が消えること）— 2p
6.3 無衝突部分: $\Theta' + \frac{ck\mu}{\mathcal H}\Theta = -\Phi' - \frac{ck\mu}{\mathcal H}\Psi$（Fourier空間、$x$ 微分形）の導出 — 2p
6.4 トムソン衝突項（静止電子→運動電子の順で導出。電子静止系での散乱の等方化＋角度依存項、実験室系へのブースト1次で Doppler 項 $\hat p\cdot\mathbf{v}_b$。結果: $C[\Theta] = -\tau'\left[\Theta_0 - \Theta + \mu v_b - \frac{1}{2}P_2(\mu)\Theta_2\right]$ ※係数 $\frac{1}{10}$ 込みの最終形は多重極展開後に提示。偏光無視による $\Pi\to\Theta_2$ の置換を明示し、誤差見積り（数%、第14章の検証で評価）を述べる。完全な角度積分は付録Eに置き、本文は等方化項・Doppler項・四重極項の3構造の導出に集中）— 5p
6.5 多重極展開と階層方程式（Legendre展開 → $\Theta_0', \Theta_1', \Theta_2', \Theta_\ell'$ の無限階層。導出契約S6.3。スライドp14と同一形であることを注記）— 3p
6.6 CDM（衝突項ゼロ、非相対論的1次モーメント2本。導出契約S6.4。「なぜCDMは2本で閉じ、光子は閉じないか」＝圧力・自由伝播の説明）— 2p
6.7 バリオン（電子・陽子をクーロン結合した単一流体として扱う正当化 → トムソン散乱の運動量交換項。**運動量保存から光子側衝突項と符号が逆で $R=\frac{4\Omega_{\gamma0}}{3\Omega_{b0}a}$ 倍**になることの導出契約S6.5）— 3p
6.8 ニュートリノ（質量ゼロ: 光子と同形で衝突項なし。質量あり: エネルギー依存性が消えず $q$ グリッドが必要になることの構造的説明のみ本文、方程式一式は付録 — 数値は第13章でCLASSに委譲することを宣言）— 1p

**導出契約**
- S6.1: 摂動計量中の測地線 → $\frac{1}{p}\frac{dp}{d\eta} = -\mathcal{H} - \Phi' - \frac{ck\mu}{?}\Psi$ 型の式（係数・符号は `03_numerics_spec.md` の最終形と整合させ、相互検算すること）。
- S6.2: トムソン衝突項の3構造（等方化・Doppler・四重極）。

- S6.5: $v_b' = -v_b - \frac{ck}{\mathcal H}\Psi + \tau' R(3\Theta_1 + v_b)$。光子・バリオン全運動量保存のチェック式を演習に。

## 第7章 初期条件と原始ゆらぎ（10p）

**目標**: 超地平線極限で階層方程式系を整合的に解き、断熱初期条件一式を導く。インフレーションは「$\mathcal{P}_\mathcal{R}(k)$ を与える機構」として最小限（4p）導入し、$A_s, n_s$ の意味を確定する。

**節構成**
