# CMBパワースペクトル教科書プロジェクト — 仕様書一式（マスター文書）

本ディレクトリは、大学院生向け教科書『ゼロから構成するCMBパワースペクトル』（本文約200ページ＋Jupyterノートブック群）を、下流AI（Codex等のコーディング/執筆エージェント）に実装させるための **完全な仕様書** である。本仕様書群のみを入力として、人間の追加判断なしに成果物が完成することを目標に書かれている。

---

## 1. プロジェクトの目的（変更不可の要求仕様）

1. **対象読者**: 物理学の大学院生（一般相対論の初歩、統計力学、電磁気学、Fourier解析を既習。宇宙論は未習でもよい）。
2. **到達目標**: 読者が FLRW計量から出発して、線形摂動論・ボルツマン方程式を**一切の論理飛躍なく**自力で追い、CMB温度パワースペクトル $C_\ell$ を自分のコードで再現し、各パラメータがスペクトルをどう変えるかを**定量的に**説明できるようになること。
3. **設計原則 — 最小構成主義**:
   - 物理的に必須でない一般化（ベクトル/テンソル摂動、ゲージの一般論の網羅、偏光の完全な取り扱い、非線形効果）は本文の主筋から外し、付録または「発展」枠に隔離する。
   - ただし「最小」とは「省略」ではない。主筋に載せた式は**全て導出する**。「示せる」「よく知られているように」で式を導入することを禁止する。
   - スカラー摂動・conformal Newtonianゲージ・温度異方性（TT）のみで $C_\ell$ 完成まで到達する。偏光は $\tau$ 測定の文脈で最小限のEモード解説を付録に置く。
4. **必須の定量トピック**（本文第13章＋ノートブックNB6で必ず扱う）:
   - ニュートリノ質量 $\sum m_\nu$ の影響
   - 光学的厚み $\tau$（再電離）の影響
   - 再結合期 $z_*$ がずれた場合の影響
   - 再電離期 $z_{\rm re}$ がずれた場合の影響
   - いずれも「どの方向に・何%・なぜ」を数式とプロットで示す。
5. **教材構成**: 本文（LaTeX→PDF, 約200ページ）＋ Jupyterノートブック9冊（NB0–NB8）＋共通Pythonパッケージ `cmbcore`。本文中の数値図版は**全て** `cmbcore` から生成し、ノートブックで読者が再現できる。

## 3. グローバル規約（全成果物で統一・違反は検収不合格）

### 3.1 記法・規約
- 計量符号: $(-,+,+,+)$。
- 摂動付き計量（conformal Newtonianゲージ、Dodelson/Callin規約。添付スライドと同一）:
  $$ds^2 = -(1+2\Psi)\,c^2 dt^2 + a^2(t)(1+2\Phi)\,\delta_{ij}\,dx^i dx^j$$
  $\Psi$: 時間成分のポテンシャル（ニュートンポテンシャルに対応）、$\Phi$: 空間曲率摂動。異方性応力が無視できるとき $\Phi \simeq -\Psi$。**Ma & Bertschinger (1995) の $(\psi,\phi)$ とは符号規約が異なる**ことを本文の記法表で明示し、対応表を付録Aに置く。
- 時間変数: 物理時間 $t$、共形時間 $\eta$（$d\eta = dt/a$）、主たる数値変数 $x \equiv \ln a$（$a_0=1$）。共形ハッブル $\mathcal{H} \equiv aH = a\,da^{-1}da/dt$、コード内表記 `Hp`。
- Fourier規約: $f(\mathbf{x}) = \int \frac{d^3k}{(2\pi)^3} \hat f(\mathbf{k}) e^{i\mathbf{k}\cdot\mathbf{x}}$。
- 温度ゆらぎの多重極展開: $\Theta(\eta,k,\mu) = \sum_\ell (-i)^\ell (2\ell+1) P_\ell(\mu)\,\Theta_\ell(\eta,k)$、$\mu = \hat k \cdot \hat p$。
- 原始スペクトル: $\mathcal{P}_\mathcal{R}(k) = A_s (k/k_{\rm pivot})^{n_s-1}$、$k_{\rm pivot} = 0.05\,{\rm Mpc}^{-1}$。
- 単位: 本文の導出は自然単位（$\hbar = c = k_B = 1$）を基本としつつ、数値実装に渡す最終形では $c$ を明示復元する（`03_numerics_spec.md` の式が正）。コードはSI。

### 3.2 基準宇宙論パラメータ（fiducial; Planck 2018 TT,TE,EE+lowE+lensing 近傍）
```
h        = 0.674
Ωb h²    = 0.0224
Ωc h²    = 0.1200
Ωk       = 0
T_CMB    = 2.7255 K
N_eff    = 3.046（質量ゼロ近似のとき）
Σmν      = 0.06 eV（既定。最小構成計算では 0 とし、第13章/NB6でのみ有効化）
n_s      = 0.965
A_s      = 2.1e-9
τ        = 0.054（瞬間再電離 z_re ≈ 7.7 相当）
ΩΛ       = 1 − (Ωb+Ωc+Ωγ+Ων)  （平坦を既定）
```
教育的単純化として、主筋の数値計算（NB1–NB5, 本文第14章まで）は「ヘリウムなし（$Y_p=0$）・質量ゼロニュートリノ・再電離なし・偏光なし」を**明示の上で**採用し、第13章/NB6でこれらを順に解除して効果を測る。この「単純化→解除」の構造自体を教育装置として使う。

### 3.3 文体・体裁
- 言語: 日本語（専門用語は初出時に英語併記）。
- 数式は全て番号付き。本文中で参照しない式にも番号を付けてよいが、導出の中間式は align 環境でまとめる。
- 各章の冒頭に「この章で導くこと（入力→出力）」、章末に「まとめ・チェックリスト・演習（3–6問、うち1問は対応ノートブックの課題）」を置く。
- 図は全章通し番号。図キャプションは図単独で意味が取れる長さ（3–6行）にする。
- 「ボックス」環境を3種定義: 【物理的直観】【数値の現実】【発展（読み飛ばし可）】。

## 5. 完成の定義（Definition of Done）

1. `cmbcore` が fiducial パラメータで $C_\ell^{TT}$（$2 \le \ell \le 1500$）を計算し、同一物理仮定の CLASS 出力と **全 $\ell$ で5%以内、$\ell \le 1000$ で3%以内** で一致する（`04_task_plan_and_review.md` §3 の検証プロトコル）。
2. 第一音響ピーク位置 $\ell_1 = 220 \pm 5$、ピーク比が検証表の範囲に入る。
3. 本文PDFが190–215ページ、全図版（約60点）が `make figures` で再生成可能。
4. ノートブック9冊（NB0–NB8）が `jupyter nbconvert --execute` でエラーなく完走（NB6のCLASS依存セルはオプションフラグで分離可）。
5. `04_task_plan_and_review.md` §5 の最終レビューチェックリスト全項目に合格。
